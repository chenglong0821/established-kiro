# Established Kiro

一个高效的学习辅助应用，帮助你管理学习时间、记录错题、笔记和练习。

## 功能特性

- **番茄钟** - 专注计时器，帮助提高专注力
- **错题本** - 记录和复习错题
- **笔记** - 整理学习笔记
- **练习** - 巩固知识点
- **统计** - 追踪学习进度

## 技术栈

- React 18
- Vite
- CSS

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
